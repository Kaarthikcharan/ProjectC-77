In computer technology, a console can mean different things depending on the context. It usually means a combination of a display monitor and an input device, usually a keyboard and mouse pair, which allows a user to input commands and receive visual output from a computer or computer system. The console is an operating system window where users interact with the operating system or with a text-based console application by entering text input through the computer keyboard, and by reading text output from the computer terminal.Right-click Start and choose Command Prompt or Command Prompt (Admin) from the Quick Link menu.
